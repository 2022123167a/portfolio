# Ahmed Sherif Portfolio

A modern personal portfolio website for Ahmed Sherif, a Junior Full-Stack Developer based in Sivas, Turkey.

## Live Preview

Add your deployed website link here after hosting on Vercel, Netlify, or another static hosting provider.

## Features

- Responsive React and Vite portfolio website
- Dark theme with subtle blue and purple accents
- Sticky navigation with mobile menu
- Hero section with profile photo, CV download, and social links
- About, skills, experience, projects, education, certifications, languages, and contact sections
- Contact form connected to Formspree
- Smooth scrolling and polished hover animations

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React
- Formspree

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Customization

- Replace `src/assets/profile.jpg` with your personal photo.
- Replace `public/Ahmed_Sherif_CV.pdf` with your final CV.
- Update project GitHub and live demo links in `src/components/Projects.jsx`.
- Update contact and social links in the relevant components.
